---
description: Test Reader Dashboard features (Feed, Read, Download, Review)
---

1. Log in as a Reader (or create a new Reader account)
2. Navigate to the Reader Dashboard
3. Verify the "New Arrivals" hero section is visible
4. Verify the list of books is displayed in a grid/feed format
5. Click the "Read" button on a book
   - Verify it navigates to the reader view
6. Click the "Save" (Download) button on a book
   - Verify the download starts (or mock alert appears)
7. Click the "Write Review" (Star icon) button
   - Verify the Review Modal appears
   - Select a star rating
   - Enter a comment
   - Click "Submit Review"
   - Verify success message/alert
